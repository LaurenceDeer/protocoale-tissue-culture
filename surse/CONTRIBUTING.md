# 🤝 Cum să Contribui

Dacă vrei să adaugi un ghid nou, o rețetă de nutrienți sau un sistem pe care l-ai construit cu succes:

1. Fă un Pull Request pe GitHub.
2. Adaugă protocolul în format Markdown.
3. Asigură-te că folosești aceleași convenții de formatare.
