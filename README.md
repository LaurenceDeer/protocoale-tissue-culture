# 🇷🇴 Grădina Ta Fără Sol 🇷🇴
**Ghidul open-source pentru sisteme hidroponice în propria casă.**

Dacă ai ajuns aici, probabil vrei să crești plante mai repede, mai sănătoase și fără murdăria asociată cu pământul obișnuit. Vrei să înveți cum să oferi plantelor nutrienții exacți de care au nevoie, direct la rădăcină.

Ai nimerit bine. Ăsta este hub-ul românesc de Hidroponie. Fără jargon academic inutil, fără paywall-uri. Doar știință aplicată direct.
